## Usage:

##### Grub2:

```bash
$ git clone https://github.com/InledGroup/ventoy-theme


$ cd ventoy-theme

$ ./install.sh
```
##### Ventoy:

* Copy the `ventoy` folder to the exFAT partition on your Ventoy USB drive, as explained [here](https://github.com/ventoy/Ventoy/issues/110).
